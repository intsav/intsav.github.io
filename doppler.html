Mitral Inflow

PROJECT INTRODUCTION

The project involves detecting E and A mitral inflow peaks using Faster-RCNN and segmentation of the mitral profile using U-net architecture


Dataset

The dataset consists of 495 images, which have annotations of E and A keypoints and binary masks for the profile.

The bounding boxes for each heartbeat are generated from keypoints. The bounding boxes are generated for heartbeats that include two beats and heartbeats that include one beat.
