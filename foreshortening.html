A4C Foreshortening

Project Description


Foreshortening in echocardiography refers to a common issue where the ultrasound beam doesn't capture the true apex of the left ventricle (LV). Instead, it cuts through a point above and anterior to the actual apex. This leads to a distorted image of the LV, making it appear shorter and rounder than its actual elongated, bullet-like shape. This project aims to create a dataset of A4C images with varying degrees of foreshortening.
